# 關於精選輯代碼

精選輯代碼（Collection Code）是一個包含精選輯資訊的代碼，設計用於快速匯出並分享給其他使用者。

每個精選輯代碼都包括版本識別碼和校驗碼，可以快速地判斷代碼版本和有效性，避免資料錯誤。

精選輯代碼的產生器由 Greatdori! 開發團隊研發。目前的產生演算法版本為 *Timbre（音）*。

## Timbre 演算法

Timbre 演算法所產生的代碼是由 94 個 ASCII 字元組成的。Timbre 產生的代碼會包含精選輯的所有卡片 ID 與精選輯的標題。您可以透過啟用「不含標題分享」來防止代碼中包含標題。

Timbre 演算法完全匿名，不會記錄卡片 ID 和精選輯標題以外的任何資料。唯一的卡片組合和名稱將會產生唯一的代碼。同時 Timbre 演算法完全離線，因此即使在離線狀態下，Timbre 演算法依然可以正常運作；Timbre 可以作為存檔安全地分享給任何人。

（註：匯入精選輯需要網際網路連線以取得卡面圖片，因此您可能無法透過 Greatdori! 離線匯入精選輯。）
